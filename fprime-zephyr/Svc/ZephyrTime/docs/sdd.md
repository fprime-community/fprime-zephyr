# Zephyr::ZephyrTime

System time provided by Zephyr.

## Requirements

| ID | Description | Verification |
|---|---|---|
| ZT-REQ-1 | The ZephyrTime component shall provide the current system time via the timeGetPort. | Verification |
| ZT-REQ-2 | The ZephyrTime component shall provide time from the Zephyr SYS_CLOCK_REALTIME clock. | Verification |
